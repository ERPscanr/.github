# Automated meta-analysis of the ERP related literature

This organization hosts a research project analyzing the topic of 
event-related potentials in the scientific literature. 

To explore this project, see the 
[project website](https://erpscanr.github.io/), and/or the 
[associated publication](https://doi.org/10.1038/s41598-022-05939-9).

This project is maintained by [Tom Donoghue](https://github.com/TomDonoghue/).
